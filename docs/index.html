How it works

The gateway bridges industrial field devices with cloud platforms using open standards.

Industrial Field Edge Server Cloud / Broker ───────────────── ─────────── ────────────── ┌──────────────┐ ┌──────────────┐ ┌────────────────┐ │ PLC / OPC UA│ ──── OPC UA ──▶│ │ ── MQTT/TLS ──▶│ AWS IoT Core │ └──────────────┘ │ Edge Gateway│ └────────────────┘ │ │ ┌──────────────┐ │ Flask Web UI│ ┌────────────────┐ │ Modbus Device│ ─── Modbus ───▶│ + Telegraf │ ─── MQTT ─────▶│ Mosquitto / │ └──────────────┘ └──────────────┘ │ Any Broker │ └────────────────┘ Browser ──▶ http://localhost:8050 (config UI)

Features

Everything you need to connect factory-floor equipment to the cloud.

OPC UA Input

Browse OPC UA server nodes, select tags, and configure polling or subscription acquisition modes.

Modbus TCP Input

Read holding registers, coils, and discrete inputs from any Modbus TCP device on the network.

MQTT Output

Publish to AWS IoT Core, Azure IoT Hub, or any MQTT broker. Supports TLS with certificate upload.

Web UI Configuration

Auto-save configuration for all inputs and outputs. No command-line editing of config files required.

Telegraf-powered

Uses InfluxData Telegraf as the data engine. The UI generates and deploys telegraf.conf automatically.

Live Dashboard

Real-time pipeline health metrics — OPC UA read counts, MQTT publish rates, CPU, memory, and disk.

MQTT Live Tail

View live MQTT messages flowing to your broker directly from the UI — no external tools needed.

Docker Native

Designed to run with docker compose up. Includes demo OPC UA server and MQTT broker for local testing.

Config Export / Import

Export and import the full gateway configuration as JSON for backup, migration, or replication.

Pipeline Dashboard

Monitor data flow from industrial devices to the cloud in real time.

Live pipeline metrics — OPC UA reads, MQTT publish rate, system health

Quick Start

Up and running in under two minutes.

  1. 1

    Clone the repository

    git clone https://github.com/cantalejo84/iiot-edge-gateway.git cd iiot-edge-gateway
  2. 2

    Start the full stack

    Includes the gateway, Telegraf, a demo OPC UA server, and a Mosquitto broker.

    docker compose up --build -d
  3. 3

    Open the web UI

    Configure your OPC UA / Modbus inputs and MQTT output, then click Deploy.

    http://localhost:8050
